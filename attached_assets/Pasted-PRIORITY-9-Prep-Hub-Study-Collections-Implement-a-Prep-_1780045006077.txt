PRIORITY 9 — Prep Hub & Study Collections

Implement a “Prep Hub” system that helps students organize and prepare study materials effectively.

Goal:
Transform the platform from a document repository into an active academic preparation workspace.

Core Features:

1. Study Collections
   Allow users to create custom preparation collections:

* Exam Prep
* Assignment Prep
* Revision Sets
* Semester Collections
* Personal Study Folders

Capabilities:

* Add/remove documents
* Reorder materials
* Add notes/descriptions
* Favorite/star collections
* Share collections optionally

2. Saved Materials & Quick Access
   Implement:

* Saved assets
* Recently viewed materials
* Continue studying section
* Quick access dashboard widgets

3. Exam Preparation Mode
   Allow students to group materials by:

* Course
* Exam date
* Semester
* Lecturer
* Topic

Optional:

* Countdown to exam
* Recommended materials
* “Most useful for this exam” ranking

4. Study Progress Tracking
   Basic preparation tracking:

* Mark document as reviewed
* Mark collection progress
* Track viewed/completed materials
* Progress percentage indicators

5. Curated Learning Paths
   Allow lecturers/admins to create:

* Recommended study paths
* Official preparation bundles
* “Start here” curated material flows

6. Smart Recommendations
   Based on:

* Course
* User activity
* Popular materials
* Related documents
* Frequently grouped assets

UI Requirements:

* Dedicated “Prep Hub” section in navigation
* Card/grid layout for study collections
* Progress indicators
* Clean academic visual style
* Strong mobile usability

Technical Requirements:

* Reuse existing asset/document architecture
* Avoid duplicating files
* Use relational linking between collections and assets
* Keep implementation modular for future AI/recommendation systems

Important:
Do NOT turn this into a full LMS.
Keep focus on:

* preparation,
* organization,
* discovery,
* academic productivity.

Goal:
Make the platform feel like an intelligent academic preparation workspace rather than only a file storage system.
