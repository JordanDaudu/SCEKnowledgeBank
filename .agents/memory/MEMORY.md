- [Storage 404 vs 500](storage-not-found-mapping.md) — missing blobs returned by getStream must be translated to notFound in the streaming layer; otherwise Chrome renders the JSON 500 body as its "blocked page" interstitial and the bug looks like a CSP/framing problem.
- [Replit GCS adapter shape](replit-gcs-adapter.md) — Replit's Object Storage uses GCS via a localhost sidecar at 127.0.0.1:1106; PRIVATE_OBJECT_DIR is "/<bucket>/<prefix>" and must be split before use; bundle externalizes `@google-cloud/*`, so the package must be a direct dep of any artifact that imports it (not just the workspace lib).
- [Seed demo conventions](seed-demo-conventions.md) — CommentReaction field is `kind` not `emoji`; ensureRequest status type includes "closed"; review-workflow docs use status+reviewReason on DocSpec; verify check count is 22.
- [Home.tsx query key pattern](home-querykey.md) — useSearchDocumentsV2 requires explicit queryKey via getSearchDocumentsV2QueryKey(params) in the query options object; omitting it causes TS2741.
